머신러닝 — 목차별 숨은 선수개념 (선수개념 진단용 지식베이스)

이 문서는 머신러닝 강의(선형대수→확률→회귀→국소학습→PCA→MLP→FCN→하이퍼파라미터→CNN→라그랑주→SVM)를 이해하는 데 필요한 "숨은 선수개념"을 목차별로 정리한 것이다. 공식 선수과목은 없지만, 아래 개념을 모르면 각 단원을 제대로 따라갈 수 없다. 핵심 선수영역은 (1) 확률론·통계학, (2) 선형대수학·공업수학, (3) 다변수 미적분학이다.

===== p1 Linear Algebra =====
숨은 선수개념: 기하학적 선형 변환(Linear Transformation), 고유값 분해(EVD) 및 고유벡터(Eigenvectors)의 물리적 의미. (강의노트 #2)
근거: 임의의 데이터 행렬을 다룰 때, 행렬 곱셈을 단순 연산이 아니라 공간을 회전하고 늘리는 '기하학적 변환'으로 인식해야 한다. 변환 후에도 방향이 바뀌지 않는 축(고유벡터)과 그 배율(고유값)을 찾는 감각이 필수적이다.
학생들이 틀리는 지점: 고유값 분해를 순수 행렬 계산 공식으로만 외우고, 선형 변환 시 축의 회전과 확장이라는 '기하학적 공간 변형'의 의미와 연결하지 못한다.
진단 질문: "임의의 행렬을 곱하는 선형 변환을 가할 때, 방향은 전혀 바뀌지 않고 오직 크기만 변하는 축(고유벡터)과 그 늘어나는 비율(고유값)의 기하학적 의미를 본인의 말로 설명해볼래요?"

===== p2 Probability =====
숨은 선수개념: 조건부 확률분포(Conditional Probability), 베이즈 정리(Bayes' Rule) 구조와 각 항의 인과적 해석, 가우시안 분포 및 독립(i.i.d.) 가정. (강의노트 #3)
근거: 파라미터 최적화에서 관측 데이터 D는 드러난 '결과'이고 가중치 w는 이를 유도한 '보이지 않는 원인'이다. 사후확률 P(w|D)를 구하는 것이 왜 결과로부터 원인을 역추적하는 인과적 과정인지 이해해야 MAP·MLE 같은 확률적 최적화의 당위성을 안다. 독립 가정은 joint 확률을 곱셈과 로그합으로 단순화하는 출발점이다.
학생들이 틀리는 지점: 베이즈 정리의 P(w|D)에서 데이터 D를 순방향 문맥으로만 착각하고, 결과물(D)로부터 보이지 않는 원인(w)을 역추적하는 인과 관계망을 그리지 못한다.
진단 질문: "베이즈 정리 식에서 데이터 D가 관측된 결과물이고 가중치 w가 이를 만들어낸 보이지 않는 원인일 때, 사후확률 P(w|D)가 왜 결과를 보고 원인을 역추적하는 과정이 되는지 설명해봅시다. 또 MAP 추정에서 분모 P(D)를 무시하고 분자만 극대화해도 되는 이유는?"

===== p3 Regression =====
숨은 선수개념: 정규방정식 유도를 위한 벡터 미분, 데이터 행렬의 열공간(Column Space)과 직교 투영(Orthogonal Projection). (강의노트 #4,5)
근거: 회귀 오차를 최소화하는 정규방정식을 유도하려면 행렬/벡터 단위 미분이 가능해야 한다. 최적 회귀선을 찾는 것이 기하학적으로 정답 벡터를 데이터 행렬의 열공간에 직교 투영하는 것과 같다는 직관이 있어야 모델의 구조적 한계를 파악한다.
학생들이 틀리는 지점: 정규방정식 유도를 기계적으로 받아들이고, 이것이 열공간에 대한 직교 투영으로 오차를 최소화하는 선형대수학적 직관임을 인지하지 못한다.
진단 질문: "선형 회귀에서 MSE를 최소화하는 해를 도출할 때, 정규방정식이 데이터 행렬의 열공간에 정답 벡터를 직교 투영하는 것과 수학적으로 어떤 연관이 있는지 설명할 수 있나요?"

===== p4 Local Learning =====
숨은 선수개념: 고차원 유클리드 공간에서의 거리(Distance Measure) 성질, 고차원 기하학에서의 부피 팽창 원리(차원의 저주). (강의노트 #6)
근거: KNN·커널 밀도 추정 같은 국소 학습은 입력 차원이 커지면 데이터 사이 거리가 모두 비슷해져 변별력이 사라진다. 이 '고차원 공간의 부피 팽창'을 기하학적으로 이해해야 거리 기반 모델의 한계를 인지한다.
학생들이 틀리는 지점: 차원이 늘면 거리가 모두 비슷해지는 '차원의 저주'를 시각적으로 이해하지 못해 거리 기반 모델의 한계를 간과한다.
진단 질문: "KNN이나 커널 밀도 추정에서 피처(차원)가 늘어날 때 데이터 사이 거리가 모두 비슷해져 변별력이 사라지는 현상을, 고차원 공간의 부피 팽창 관점에서 설명해볼래요?"

===== p5 PCA =====
숨은 선수개념: 다차원 공분산 행렬(Covariance Matrix) 계산 및 통계적 의미, 대칭 행렬의 직교 고유분해(Spectral Theorem). (강의노트 #7)
근거: 피처 간 상관관계를 담은 공분산 행렬은 대칭 행렬이다. 대칭 행렬이 서로 직교하는 고유벡터로 분해된다는 정리(Spectral Theorem)를 알아야, 분산이 가장 큰 축을 찾아 정보 손실 없이 차원을 축소하는 PCA 메커니즘을 증명할 수 있다.
학생들이 틀리는 지점: 공분산 행렬을 숫자가 가득 찬 표로만 보고, 고유분해로 최대 분산 방향(고유벡터)과 퍼짐(고유값)을 추출하는 공간적 의미를 놓친다.
진단 질문: "고차원 공분산 행렬을 고유값 분해하는 것이, 데이터 분포의 최대 분산 방향(주축)을 찾아 정보 손실을 최소화하며 차원을 축소하는 PCA와 어떻게 연결되는지 기하학적 당위성을 설명해봅시다."

===== p6 MLP =====
숨은 선수개념: 선형 분리 가능성(Linear Separability)의 수학적 한계, 비선형 사상(Non-linear Mapping)을 통한 공간 왜곡 원리. (강의노트 #8,9,10,11)
근거: 단층 퍼셉트론은 XOR 같은 선형 분리 불가능 문제를 못 푼다. 선형 변환(행렬 곱) 사이에 비선형 활성화 함수를 끼워 고차원 공간을 접고 펴서 결정 경계를 직선으로 만드는 원리(공간 왜곡)를 이해해야 한다.
학생들이 틀리는 지점: 층을 깊게 쌓기만 하면 다 풀린다고 생각하고, 비선형 활성화가 공간을 어떻게 변형해 결정 경계를 만드는지 당위성을 놓친다.
진단 질문: "선형 분리가 불가능한 데이터를 풀기 위해 퍼셉트론을 여러 층으로 쌓고 사이에 비선형 활성화 함수를 넣는 것이, 공간을 수학적으로 어떻게 변형(Non-linear Mapping)시키는지 말해볼래요?"

===== p7 FCN =====
숨은 선수개념: 다변수 합성함수의 미분(Chain Rule), 야코비안 행렬(Jacobian)을 포함한 행렬/벡터 미분. (강의노트 #12,13)
근거: 완전연결 신경망은 수많은 가중치가 얽힌 거대한 다변수 합성함수다. 출력층 에러 그라디언트가 입력층까지 끊기지 않고 전달되는 역전파를 이해하려면 야코비안 포함 벡터 미분과 연쇄 법칙의 수식 전개가 밑바탕이어야 한다.
학생들이 틀리는 지점: 역전파를 고정된 정적 공식으로 암기하고, 연쇄 법칙이 에러 그라디언트를 입력층까지 동적으로 전파하는 메커니즘을 유도하지 못한다.
진단 질문: "다변수 합성함수인 FCN에서 연쇄 법칙(Chain Rule)이 어떻게 출력층의 에러 그라디언트를 입력 레이어까지 전파시키는지 그 과정을 수식적으로 설명해봅시다."

===== p8 Hyperparameter =====
숨은 선수개념: 편향-분산 트레이드오프(Bias-Variance)의 오차 분해 수식, L1/L2 노름과 규제(Regularization)의 통계학적 의미. (강의노트 #14)
근거: 복잡도를 무작정 높이면 학습 오차는 줄지만 일반화가 나빠진다. 총오차가 편향²+분산+노이즈로 분해되는 원리를 알아야 하고, L1/L2 규제가 가중치 공간에 제약을 걸어 과대적합을 막는 통계적 원리를 이해해야 튜닝이 가능하다.
학생들이 틀리는 지점: 과대적합을 "학습 데이터에만 맞춰진 상태"로만 외우고, MSE = Bias² + Variance 분해와 규제가 분산을 제어하는 원리를 짚지 못한다.
진단 질문: "복잡도가 증가할 때 전체 오차가 편향과 분산의 합으로 분해되는 원리를 바탕으로, 과대적합 상태에서 규제(Regularization)를 거는 것이 수학적으로 어떤 역할을 하는지 설명해볼래요?"

===== p9 CNN =====
숨은 선수개념: 2차원 이미지의 합성곱(Convolution) 연산, 공간적 불변성(Invariance)과 등가성(Equivariance)의 대수학적 개념. (강의노트 #15,16,17)
근거: 공간 구조가 중요한 이미지에 FCN을 그대로 쓰면 파라미터가 폭발한다. 2D 합성곱의 수학적 정의를 알아야, 필터 공유와 국소 수용장이 어떻게 파라미터를 줄이면서 공간적 불변성을 얻는지 이해한다.
학생들이 틀리는 지점: 이미지에 FCN을 쓰면 왜 파라미터 폭발이 나는지 차원 관점에서 계산하지 못하고, 합성곱이 커널 공유로 불변성·등가성을 유지하는 이점을 놓친다.
진단 질문: "이미지에 FCN을 그대로 쓰면 생기는 파라미터 폭발을, CNN의 '가중치 공유'와 '국소 수용장'이 수학적으로 어떻게 제어하는지 공간 연산 특성과 연계해 설명할 수 있나요?"

===== p10 Lagrange Multiplier =====
숨은 선수개념: 등고선(Level Set)과 그라디언트의 기하학적 직교성, 부등식 제약 하의 KKT 조건 기초. (강의노트 #18)
근거: 현실 최적화는 제약 조건이 따라붙는다. 제약 경계선과 목적함수의 등고선이 접하는 지점에서 두 함수의 그라디언트가 평행해진다는 기하학적 원리를 그릴 수 있어야 한다.
학생들이 틀리는 지점: 미분해서 0이 되는 지점만 찾는 일반 최적화에 갇혀, 제약 하에서 그라디언트가 평행해지는 원리를 받아들이기 어려워한다.
진단 질문: "등식/부등식 제약이 걸린 공간에서 제약선과 목적함수 등고선이 접하는 지점을 찾는 라그랑주 승수법의 기하학적 기초(그라디언트 벡터의 관계)를 설명해봅시다."

===== p11 SVM =====
숨은 선수개념: 초평면(Hyperplane) 방정식과 법선 벡터의 직교성, 최적화의 Primal-Dual 변환(Strong Duality), 내적 공간과 커널 트릭. (강의노트 #19)
근거: 두 클래스를 가르는 초평면에서 가중치 w가 결정 경계에 법선(직교)인 이유를 유도해야 마진 수식을 세운다. 원 문제(Primal)를 쌍대(Dual)로 뒤집어야 데이터의 내적 공간 성질을 활용해 고차원 매핑을 연산 없이 처리하는 '커널 트릭'을 쓸 수 있어, 강쌍대성이 필수다.
학생들이 틀리는 지점: w^T x + b = 0에서 w가 왜 경계면의 법선 벡터인지 설명 못 하고, Dual로 뒤집어야 커널 트릭을 쓸 수 있다는 연계성을 놓친다.
진단 질문: "SVM 결정 경계 w^T x + b = 0에서 법선 벡터 w와 경계면의 기하학적 관계는 무엇이며, 왜 이 문제를 라그랑주 Dual로 전환해야 커널 트릭을 쓸 수 있는지 내적 공간 성질과 연계해 설명해볼래요?"

===== p12 손실함수 (MSE vs CEE) =====
숨은 선수개념: MSE와 CEE의 수학적 정의, 분류에서 sigmoid/softmax와 결합 시 gradient 유도.
근거·틀리는 지점: 분류 문제에서 sigmoid+MSE를 쓰면 예측이 크게 틀려도 gradient가 0에 가까워져 학습이 정체된다. Cross-Entropy(CEE)를 쓰면 이 gradient 소실이 해결되는 이유를 수식으로 이해해야 한다.
진단 질문: "분류에서 출력층 sigmoid 뒤 MSE를 쓰면 오차가 큼에도 학습이 극도로 느려지는 원인을 활성화 함수의 미분값과 연관지어 설명하고, CEE를 쓰면 왜 이 문제가 해결되는지 수식으로 풀어볼래요?"

===== p13 역전파와 경사 소실 =====
숨은 선수개념: 체인 룰 기반 역전파, 활성화 함수 미분값 누적에 따른 경사 소실(Gradient Vanishing)과 ReLU.
근거·틀리는 지점: 역전파가 체인 룰로 gradient를 입력층까지 전파하는 역동적 과정을 정적 공식으로만 받아들인다. 시그모이드 계열 미분값이 반복 곱해져 경사 소실이 나는 원인과 ReLU 도입 이유를 이해해야 한다.
진단 질문: "깊은 신경망에서 시그모이드 활성화를 쓸 때 체인 룰 때문에 경사 소실이 발생할 수밖에 없는 수학적 이유와, 이를 해결하는 대안(예: ReLU)을 말해볼래요?"

===== p14 Backfill 순서 (계단식 학습 경로) =====
[1단계 · 선형대수학 및 기하] 벡터 공간과 초평면 방정식 → 고유값·고유벡터의 기하학적 의미(선형 변환과 주축) → 다차원 공분산 행렬의 고유값 분해를 통한 데이터 분포 해석 → SVM 마진 수식화·기하 해석.
[2단계 · 베이지안 기초와 다차원 확률분포] 조건부 확률과 베이즈 정리 → 가우시안 분포의 수학적 형태와 독립 가정 하 Likelihood 곱셈·로그합 변환 → 데이터(D)와 가중치(w)의 결과-원인 역추적 시각화 → 차원의 저주 극복을 위한 가설 학습.
[3단계 · 모델 복잡도와 손실함수 최적화] Bias-Variance Trade-off와 MSE = Bias² + Variance 분해 → L1/L2 규제·오버피팅 방지의 통계 원리 → MSE/CEE 정의 및 분류에서 sigmoid-CEE 결합 시 gradient 유도.
[4단계 · 다변수 체인 룰과 역전파·경사 소실] 다변수 합성함수 미분 → 다중 MAP·Loss 최적화용 벡터 미분 → 고차원 입력에 따른 역전파 전개 → 활성화 미분값 누적에 따른 경사 소실과 ReLU.
